# Projeto de Estudo de Caso e implementação de sistema de informação para a empresa Re.Nascer Soluções em Resíduos Plásticos Ltda. 

`CURSO: Sistemas de Informação`

`DISCIPLINA: Projeto - Trabalho de Conclusão de Curso`

`SEMESTRE: 8º`

O projeto consiste em um estudo de caso e implementação de um sistema de informação para a empresa Re.Nascer Soluções em Resíduos Plásticos Ltda., uma microempresa de Itaúna-MG que atua na reciclagem e beneficiamento de resíduos plásticos. O trabalho tem como objetivo principal propor uma solução tecnológica que substitua os controles manuais atuais, baseados em anotações em papel e planilhas, por um sistema integrado de gestão, capaz de automatizar e otimizar os processos de coleta, recebimento, estoque e vendas de materiais recicláveis.

A proposta inclui o desenvolvimento de um sistema web com módulos de cadastro, agendamento de coletas, controle de recebimento, vendas e dashboards gerenciais, visando melhorar a confiabilidade dos dados, reduzir retrabalhos e fornecer informações estratégicas para a tomada de decisão. O projeto está estruturado em etapas que envolvem análise de mercado, mapeamento de processos, plano de inteligência competitiva, especificação de requisitos e modelagem inicial da solução, buscando alinhar práticas sustentáveis, eficiência operacional e conformidade com normas como a LGPD.
